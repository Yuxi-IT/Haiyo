# 目的：

解决独居人群的 办公、陪伴、家居、健康 需求

# 功能：

实时人体跟踪（舵机+摄像头）、

人体姿态检测(机器学习)，并分析人体安全（摔倒、躺/趴姿势不当）、

基于 肤色/表情/体温检测 简单分析生理状态、

语音情感陪伴、语音对话、

软件端：支持通过 语音转文字 发送消息给AI，进行电脑操控、

接入微信、飞书Bot

# 场景：

办公、

## 健康： 心理：1.文字记录AI引导提问 2.情绪识别收纳（解决方式：AI医生

情绪记录定格瞬间：通过传感器记录当下个人的地理位置坐标、周围环境的温湿度情况，记录下当下每个人的心情，每个人的数据以单个星的形式最终汇总成为一片情绪星云。（硬件实现：构建黑箱结构）



# 所具备的硬件：

摄像头

舵机

红外传感器

温度传感器

湿度传感器
